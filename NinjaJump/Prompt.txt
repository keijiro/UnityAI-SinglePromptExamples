忍者が主人公の2Dジャンプアクション＋infinite runner 系カジュアルゲームを作成して下さい。
- 主人公は忍者で、何も操作しなくても常に画面右方向に走っています。
- ただし、画面上での位置は固定で、周囲がスクロールしている、という格好です。
- 上キーを押すことでバク転ジャンプします。その間も同じ速度で移動は続けます。
- ジャンプすると同時に刀を突き出します。突き出した状態のままバク転するため、刀で斬る格好になります。
- 右キーを押している間少しだけ速度が増し、左キーを押している間少しだけ速度が減ります。
- 画面右からは敵忍者が現れます。発生間隔はランダムです。
- 敵忍者は、地面に静止しているものと、ジャンプを繰り返すものと、２種類あります。
- 敵忍者に刀が当たると、敵忍者は爆発して消え、得点を得ます。
- 敵忍者を倒さずに通過させてしまうと、得点を失います。
- 敵忍者に自機が触れるとゲームオーバーです。
- スプライト素材は適切に透過されるようにして下さい。
- UI には UI Toolkit を使用します。既に配置してある UI Document を書き換えて使って下さい。
