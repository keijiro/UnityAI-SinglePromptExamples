牧羊犬をテーマとした 2D カジュアルゲームを作成します。
- ステージ上にはコーギー犬が３匹ランダムな位置に配置されています。
- プレイヤーはこれらのコーギー犬をマウスドラッグで移動できます。
- 画面下のランダムな位置から羊が１列に連なって画面上方向へと流れてきます。
- 画面上端中央に小屋が存在します。
- 羊が小屋に到達すると得点になります。
- 羊はランダムな匹数（１５匹程度）出現した後、また別のランダムな位置から出現するのを繰り返します。
- 羊はコーギー犬に近づくとそれから逃げるような動きをします。
- 画面上にはランダムに２匹の狼が出現します。
- 狼は数十秒同じ位置に静止し、その後いなくなり、また別の狼が現れて、を繰り返します。
- 羊は狼にぶつかるとやられてしまい、原点となります。
- アートスタイルは可愛く柔らかなイラスト調を基調として下さい。
- スプライト素材は適切に透過されるようにして下さい。
- UI には UI Toolkit を使用します。既に配置してある UI Document を書き換えて使って下さい。
